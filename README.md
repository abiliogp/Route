## Goals
* Clean Code

* Dijkstra Algorithm - Swift

* Autocomplete for TextField

* MVVM

* Localizable Strings

* No External Libs

* Base URL stored xcconfig

* Unit test for MVVM

* Unit test for ApiService

* Unit test for Dijkstra Algorithm

* TDD (Test-driven development)
